standard.net.ftp.FtpQuit

Name Version Deprecated
FtpQuit version 1.0 no

Prototype:
null-returned FtpQuit(number connection_handle)

Parameters:
connection_handle      the connection handle as returned by FtpConnect.

Description:
Disconnect from remote.

Example: 
	function TestFtp() {
		var con=FtpConnect("ftp.rdsct.ro");
		if (!con) {
			echo "Can't Connect\n";
			return 0;
		} else {
			echo "Connected !!!\n";
		}
		FtpLogin("","",con);
		//if (!FtpGet("c:\\test.zip","/pub/PPPoE/win98/RASPPPOE_098B.ZIP",FTPLIB_IMAGE,con))
		//	echo "Error getting file\n";
		var ftpfile=FtpAccess("/pub/PPPoE/win98/RASPPPOE_098B.ZIP",FTPLIB_FILE_READ,FTPLIB_IMAGE,con);
		if (ftpfile) {
			echo "File opened !\n";
			echo FtpRead(1,ftpfile);
			FtpClose(ftpfile);
		}
		FtpQuit(con);
	}

Returns:
Returns nothing
