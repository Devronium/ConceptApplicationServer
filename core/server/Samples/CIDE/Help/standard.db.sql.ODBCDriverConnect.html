standard.db.sql.ODBCDriverConnect

Name Version Deprecated
ODBCDriverConnect version 1.0 no

Prototype:
number ODBCDriverConnect(string ConnectionString)

Parameters:
ConnectionString      the ODBC connection string to use

Description:
Connects to a database using a connection string. You may want to check ODBCConnect.

Example of connection strings:

MSSQL:Driver={SQL Server};Server=localhost;Database=pubs;Uid=;Pwd=
Access:Driver={Microsoft Access Driver (*.mdb)};Dbq=mydatabase.mdb;Uid=;Pwd=
Oracle:Driver={Microsoft ODBC for Oracle};Server=OracleServer.world;Uid=;Pwd=

Interbase:Driver={Easysoft IB6 ODBC};Server=localhost;Database=localhost:mydatabase.gdb;Uid=username;Pwd=password
IBM DB2:Driver={IBM DB2 ODBC DRIVER};Database=myDbName;hostname=myServerName;port=myPortNum;protocol=TCPIP; uid=myUserName; pwd=myPwd
Sysbase 12:Driver={SYBASE ASE ODBC Driver};Srvr=localhost;Uid=username;Pwd=password:
Sysbase 11:Driver={SYBASE SYSTEM 11};Srvr=localhost;Uid=username;Pwd=password;Database=mydb:
Informix:Dsn='';Driver={INFORMIX 3.30 32 BIT};Host=hostname;Server=myserver;Service=service-name;Protocol=olsoctcp;Database=mydb;UID=username;PWD=myPwd

Mimer SQL:Driver={MIMER};Database=mydb;Uid=myuser;Pwd=mypw

Returns:
Returns null if failed, or the connection id if succeeded.
