standard.lib.regex.regex

Name Version Deprecated
regex version 1.0 no

Prototype:
array regex(string string_to_parse, string pattern)

Parameters:
string_to_parse      the subject string
pattern      the pattern

Description:
Split string into array by regular expression. Returns an array of strings, each of which is a substring of string_to_parse formed by splitting it on boundaries formed by the case-sensitive regular expression pattern.

Returns:
Returns an array of strings, each of which is a substring of string_to_parse formed by splitting it on boundaries formed by the case-sensitive regular expression pattern.
