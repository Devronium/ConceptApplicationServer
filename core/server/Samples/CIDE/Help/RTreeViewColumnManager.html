RTreeViewColumnManager

Name Version Deprecated
RTreeViewColumnManager version 1.0 no

Defined in : RTreeViewColumnManager.con
(this file must be included in your application in order to have access, unless you use a child class of RTreeViewColumnManager that includes this file)

Description:
RTreeViewColumnManager is a class that provides some kind o virtual access to the entire columns, without creating objects for each column. This class should not be used directly; it's used internally by the RTreeView control.

Concept, as you know, is an application server, so it runs applications. In doing this, it consumes memory and for that needs to be as memory efficient as possible.


Methods:
PostString This function posts a string to the client
RTreeViewColumnManager This is the standard contructor for this class and should not be used directly
operator[] This operator provides access to the virtual columns (of the RTreeView)
