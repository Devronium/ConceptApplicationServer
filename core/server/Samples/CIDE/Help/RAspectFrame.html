RAspectFrame

Name Version Deprecated
RAspectFrame version 1.0 no

Defined in : RAspectFrame.con
(this file must be included in your application in order to have access, unless you use a child class of RAspectFrame that includes this file)

Inherits : RFrame  

Description:
A frame that constrains its child to a particular aspect ratio.

The RAspectFrame is useful when you want add a control so that it can resize but always retains the same aspect ratio. For instance, one might be drawing a small preview of a larger image. Because it derives from RFrame, it can draw a label and a frame around the child. The frame will be "shrink-wrapped" to the size of the child.


Properties:
ObeyChild If true then force aspect ratio to match that of the frame's child
Ratio Sets or gets the aspect ratio
XAlign Sets or gets the horizontal alignment of the child
YAlign Sets or gets the vertical alignment of the child
