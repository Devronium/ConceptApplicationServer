standard.C.string.memcmp

Name Version Deprecated
memcmp version 1.0 no

Prototype:
number memcmp(string buf1, string buf2, number count)

Parameters:
buf1      First buffer.
buf2      Second buffer.
count      Number of bytes.

Description:
Compare characters in two buffers. Compares the first count bytes of buf1 and buf2 and returns a value indicating their relationship.

Returns:
The return value indicates the relationship between the buffers. Returns: < 0 if buf1 is less than buf2 0 if buf1 is identical to buf2 > 0 if buf1 is greater than buf2
