Require : standard.lib.face

Description:
Face/object detection and camera functions (server-side).


Contained static functions:
CameraFrame Gets a frame from the given camera in szParam2 format
CloseCamera Gets a frame from the given camera in szParam2 format
CloseObject Closes an object opened with LoadObject
CloseRecognize Close a recognize handle opened by FaceRecognize
FaceRecognize Recognizes a face and returns its index
LoadObject Loads xml file describing an object
ObjectDetect Detects an object and returns an array
OpenCamera Opens camera number nParam1
ReadCamera Gets a frame from the given nParam1 camera index after a delay of nParam3 milliseconds
