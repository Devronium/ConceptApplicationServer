standard.db.sql.ODBCListTypes

Name Version Deprecated
ODBCListTypes version 1.0 no

Prototype:
number ODBCListTypes([number connectionID])

Parameters:
connectionID      the ID identifying the connection as returned by ODBCDriverConnect or ODBCConnect. If this parameter is missing, this will affect the first connection created.

Description:
Get information about data types supported by the data source. The result is stored as a result set.

Returns:
Return a result set handle, or null if failed.
