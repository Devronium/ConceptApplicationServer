standard.db.sql.ODBCUnlock

Name Version Deprecated
ODBCUnlock version 1.0 no

Prototype:
boolean ODBCUnlock(number statement)

Parameters:
statement      [in] the statement handle (or result set) as returned by ODBCStatement, ODBCListTables, ODBCListColumns, ODBCListTypes or ODBCExecuteQuery

Description:
Unlock the access to a previously locked record. Records can be locked by calling ODBCLock.

Returns:
Return true if succeeded, false if failed.
